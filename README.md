# Anime Character Creator 🥷

A Python program that generates a custom anime character card based on your name, age and fighting style.

## What it does
- Takes your name, age and preferred fighting style as input
- Calculates your power level
- Assigns you a rank (Genin, Chunin or Kage)
- Outputs your full character card

## Ranks
| Power Level | Rank |
|-------------|------|
| Below 100 | Genin |
| 100 - 499 | Chunin |
| 500+ | Kage |

## How to run
```bash
python power.py
```

## Built with
- Python 3
- Functions, f-strings, if/elif/else
